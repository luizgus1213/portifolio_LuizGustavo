Sistema web de loja online desenvolvido com Next.js, React, TypeScript, Node.js, Sequelize e SQLite.

O projeto foi criado com foco em simular uma loja virtual funcional, com cadastro de usuários, autenticação, produtos, imagens, carrinho, pedidos, perfil do usuário e recursos administrativos.

- Autenticação com token e cookies

- Verificação de email
- Cadastro de produtos
- Upload de imagens
- Listagem e pesquisa de produtos
- Página individual de produto
- Carrinho de compras
- Checkout
- Sistema de pedidos
- Histórico de compras
- Favoritos
- Cupons

- Perfil do usuário
